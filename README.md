## âš¡ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn
- Google Chrome browser

### Installation

```bash
# Step 1: Clone the repository
git clone https://github.com/chater-marzougui/beautiful-notes-extension.git

# Step 2: Navigate to the project directory
cd beautiful-notes-extension

# Step 3: Install dependencies
npm install

# Step 4: Build the extension
npm run build
```

**Load into Chrome:**
1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode" in the top right.
3. Click "Load unpacked".
4. Select the `dist` folder created by the build process.

## ğŸª› Configuration

The extension requires a Google Gemini API key for AI features.

1. Get a free API key from [Google AI Studio](https://aistudio.google.com/).
2. Enter it in the extension settings.
3. The key is stored securely in your local Chrome storage and is never sent to any other server.
